Dacorweb - Sistema ERP

Software gerenciador para lojas e pontos de venda (PDV), apresentado como Trabalho de Conclusão de Curso (TCC).

GitHub Ver Projeto

01. Sobre o Projeto

A Dacorweb, é Trabalho de Conclusão de Curso (TCC) apresentado em 2023 no IFC sendo avaliado com nota máxima (10), tem como propósito a criação de um software abrangente para o gerenciamento e controle de vendas, estoque, usuários e geração de relatórios. Vale ressaltar que o sistema é caracterizado por um layout simples e funcional, incorporando um painel de controle (dashboard) e gráficos para proporcionar uma experiência completa e intuitiva, além dos níveis separados por sessões para cada usuário.

02. Principais Funcionalidades

Painel Administrador

  • ▹Cadastro: Funcionários, Fornecedores, Formas de Pagamento, Clientes.
  • ▹Produtos: Cadastrar Produtos e Categorias, Listar Compras, Estoque Baixo, Alertas de Vencimento.
  • ▹Vendas: Listar todas as vendas realizadas.
  • ▹Relatórios: Geração de relatórios completos de Produtos, Compras, Vendas, Contas a Pagar/Receber e mais.

Painel Financeiro

  • ▹Contas: Gestão de contas a pagar e receber, com filtros por vencimento.
  • ▹Movimentações: Histórico completo de transações financeiras.
  • ▹Relatórios Financeiros: Relatórios detalhados de movimentações e contas.

Painel Operador (PDV)

  • ▹Interface otimizada para Ponto de Venda (PDV).
  • ▹Funções rápidas para passar produtos, consultar preços, fechar vendas e gerar cupons.
